# 📦 Tracking Number Generator API

A RESTful API built with Spring Boot 3.5 to generate unique, scalable tracking numbers for parcels.

## 🔧 Tech Stack
- Java 17
- Spring Boot 3.2
- Gradle
- Spring Web, Validation
- SpringDoc OpenAPI

./gradlew bootRun
